🚗 ОПИСАНИЕ ПРОЕКТА: ИНТЕРНЕТ-МАГАЗИН MERCEDES-BENZ

Данный проект представляет собой мини-интернет-магазин автомобилей Mercedes-Benz, разработанный с использованием HTML, CSS и JavaScript. Сайт имитирует работу настоящего онлайн-каталога с возможностью выбора автомобилей, добавления их в корзину и оформления покупки.

━━━━━━━━━━━━━━━━━━━━━━━
🏠 ГЛАВНАЯ СТРАНИЦА (КАТАЛОГ)
━━━━━━━━━━━━━━━━━━━━━━━

На главной странице представлен каталог автомобилей Mercedes-Benz:

• Mercedes C-Class  
• Mercedes G-Class  
• Mercedes AMG GT  

Каждая карточка автомобиля включает:
✔ изображение автомобиля  
✔ краткое описание  
✔ цену  
✔ кнопку «Подробнее»  

При нажатии на кнопку «Подробнее» пользователь переходит на отдельную страницу выбранной модели, где представлена подробная информация.

━━━━━━━━━━━━━━━━━━━━━━━
🚘 СТРАНИЦЫ АВТОМОБИЛЕЙ
━━━━━━━━━━━━━━━━━━━━━━━

Каждый автомобиль имеет отдельную страницу с расширенным описанием.

На странице отображается:
✔ крупное изображение автомобиля  
✔ описание модели  
✔ технические характеристики  
✔ цена  
✔ кнопка «Добавить в корзину»  

При добавлении автомобиля в корзину данные сохраняются в localStorage браузера.

━━━━━━━━━━━━━━━━━━━━━━━
🛒 КОРЗИНА ПОКУПОК
━━━━━━━━━━━━━━━━━━━━━━━

Корзина реализована с использованием localStorage, что позволяет сохранять товары даже после перезагрузки страницы.

Функционал корзины:
✔ отображение добавленных автомобилей  
✔ отображение названия, изображения и цены  
✔ возможность удаления товаров  
✔ автоматический подсчёт общей стоимости  

━━━━━━━━━━━━━━━━━━━━━━━
⚙️ ЛОГИКА JAVASCRIPT
━━━━━━━━━━━━━━━━━━━━━━━

В проекте реализованы основные функции:

➕ addToCart()
Добавляет выбранный автомобиль в корзину и сохраняет его в localStorage.

📦 loadCart()
Загружает все товары из localStorage и отображает их на странице корзины, одновременно рассчитывая общую стоимость.

❌ removeItem()
Удаляет выбранный товар из корзины и обновляет данные.

━━━━━━━━━━━━━━━━━━━━━━━
💳 ОФОРМЛЕНИЕ ЗАКАЗА
━━━━━━━━━━━━━━━━━━━━━━━

После нажатия кнопки «Купить» пользователь попадает на страницу подтверждения заказа.

На странице отображается:
✔ сообщение об успешной покупке  
✔ подтверждение оформления заказа  
✔ кнопка возврата в каталог
